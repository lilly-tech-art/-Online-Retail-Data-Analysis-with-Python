This project was completed as part of my Week 1 Internship at Analyst Lab. The objective was to clean, explore, and analyze an online retail dataset using Python to uncover meaningful business insights.
Objectives
Clean and preprocess the dataset.
Handle missing values and duplicate records.
Convert data into appropriate formats.
Perform Exploratory Data Analysis (EDA).
Visualize key trends and patterns.
Generate actionable business insights.

Tools & Technologies
Python
Google Colab
Pandas
NumPy
Matplotlib

Dataset
The project uses the Online Retail Dataset, which contains transactional data from a UK-based online retailer between December 2010 and December 2011.
📊 Project Workflow
Imported the dataset.
Inspected the data for quality issues.
Cleaned missing values and duplicates.
Converted the invoice date to datetime format.
Performed Exploratory Data Analysis (EDA).
Created visualizations to identify sales patterns and top-selling products.
Summarized findings and business insights.
🔍 Key Insights
Identified the top-selling products based on quantity sold.
Explored customer purchasing patterns.
Highlighted the importance of data cleaning before analysis.
Generated visual insights to support business decision-making.
